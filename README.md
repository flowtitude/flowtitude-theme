# Flowtitude - Bricks Builder Child Theme


### Warning: This theme is tested only as a child theme of Bricks Builder, although it can be used with slight modifications for other visual builders.


## Theme Settings

Flowtitude theme adds theme settings and lots of features with it. 


### SNN Settings & Features

- Remove/Hide WP Version	
- Disable XML-RPC	
- Disable Theme/Plugin File Editing 
- Disable Remove RSS	
- Disable JSON API for Guests
- Tailwind Scanner for Child Theme Files
- Disabling Yabe SIUL update to WindPress
- Creating images for dynamic placeholders with light and dark theme

### Move Bricks Menu to End	

Enabling this setting will move the Bricks menu item to the end of the WordPress admin menu.

### Add or disable features

You can easily add or disable features without having to edit any files, just place the snippet in the /inc/settings or inc/custom directory, it will automatically be added to the theme, you can temporarily disable it by adding an underscore at the beginning of the name, this way of working works for other files such as dynamic tags, conditionals (coming soon) or custom elements.

### Add CSS or JS to the child theme

In the same way you can add or temporarily disable CSS or JS files and they will automatically be enqueued in your child theme, you just have to add it to the assets/css and assets/js directories.
